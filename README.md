A modern, interactive **Simon Says memory game** built using **HTML, CSS, and JavaScript**.  
Includes difficulty levels, sound toggle, scoring system, animations, and high score storage.


##  Features

-  Clean modern UI  
-  Difficulty modes: **Easy, Medium, Hard**  
-  Sound ON/OFF toggle  
-  Sequence animations  
-  Score + High Score (saved in browser)  
-  Error detection with restart  
-  Fully responsive

  ##  How to Play

1. Click **Start Game**  
2. Watch the sequence carefully  
3. Repeat it by clicking colored boxes  
4. Each round adds a new step  
5. Game ends when you tap the wrong color  


##  Difficulty Levels

| Easy   | Slow |
| Medium | Faster |
| Hard   | Fastest |

---

##  Sound System
You can toggle sound ON/OFF anytime.
